## 시간 배분

### 1부: 개념 소개 (25분)

**[0:00~0:10] AI란 무엇인가?**
- "AI = 사람이 규칙을 짜는 게 아니라, 컴퓨터가 스스로 규칙을 찾는 것"
- AI 학습 방법 3가지 소개: 지도학습, 비지도학습, **강화학습/진화학습**
- 오늘의 주제: 유전 알고리즘 = "진화를 모방한 최적화"

**[0:10~0:25] 유전 알고리즘 핵심 개념**
- 생물 진화 비유로 설명:
  - **개체(Individual)** = 자동차 1대 (신경망 가중치가 "유전자")
  - **인구(Population)** = 자동차 N대
  - **적합도(Fitness)** = 보상 함수 점수 (얼마나 잘 주행했나)
  - **선택(Selection)** = 점수 높은 자동차가 "부모"
  - **교차(Crossover)** = 부모 둘의 가중치를 섞어 자식 생성
  - **변이(Mutation)** = 가중치에 랜덤 노이즈 추가
  - **세대(Generation)** = 한 바퀴 돌고 다음 세대 생성
- 핵심 질문: "왜 정답을 가르치지 않아도 점점 잘하게 되는가?"

### 2부: 시뮬레이터 탐색 (20분)

**[0:25~0:35] 시뮬레이터 실행 및 UI 소개**
- 함께 실행하면서 설정 화면 안내
- 기본 설정으로 한 번 돌려보기 (자동차 10대, 센서 5개)
- 우측 패널의 그래프와 GA 정보 설명

**[0:35~0:45] 코드 구조 설명 (핵심만)**
- `CarBrain` 클래스: 신경망 구조 (입력→은닉→출력)
  - "이 신경망이 자동차의 '두뇌'입니다"
  - "센서 데이터를 받아서 좌/우 조향 결정"
- `mutate()`: 가중치에 노이즈 추가 → 돌연변이
- `crossover()`: 두 부모의 가중치를 50:50으로 섞기
- `compute_fitness()`: 보상 함수 계산
- `next_generation()`: 선택 → 교차 → 변이 → 새 세대

### 3부: 실험 실습 (45분) ⭐ 핵심

**[0:45~1:05] 실험 1: 보상 함수 비교** (20분)
- 시드를 42로 고정

- **토론**: "어떤 보상 함수가 가장 빨리 완주하나?"

**[1:05~1:15] 실험 2: 인구 크기의 영향** (10분)

### 4부: 토론 및 정리 (30분)

**[1:30~1:45] 실험 결과 공유**
- CSV 로그 파일 열어서 세대별 적합도 추이 확인
- 각자 다른 실험 결과를 공유
- 가장 빨리 완주한 설정은?

**[1:45~1:55] 핵심 개념 정리**
- GA vs 경사 하강법 (Gradient Descent) 비교
  - GA: 미분 불필요, 탐색 범위 넓음, 느림
  - GD: 미분 필요, 정밀, 빠름
- "보상 함수 설계가 AI의 행동을 결정한다" → AI 정렬(Alignment) 문제 연결
- 현실 응용: 로봇 제어, 게임 AI, 안테나 설계, 물류 최적화

**[1:55~2:00] Q&A 및 마무리**

## 자주 묻는 질문 (FAQ)

**Q: 왜 역전파를 안 쓰나요?**
A: 역전파는 손실 함수가 미분 가능해야 합니다. 자동차가 벽에 부딪히는 건 미분할 수 없는 이산적 이벤트라서, 대신 GA로 "시행착오"를 통해 학습합니다.

**Q: 왜 매번 결과가 다른가요?**
A: 시드가 다르면 초기 가중치와 변이 방향이 달라집니다. 같은 시드를 쓰면 재현됩니다.

**Q: 자동차가 벽에 갇혀서 안 움직여요**
A: 정체(stagnation) 감지가 60스텝마다 작동합니다. 정체된 차는 주황색으로 표시되고 해당 세대가 빨리 종료됩니다.

**Q: 보상 함수를 여러 개 선택하면?**
A: 합산됩니다. 예를 들어 "최대 거리 + 벽 회피"를 선택하면 멀리 가면서 벽을 피하는 행동을 학습합니다.
